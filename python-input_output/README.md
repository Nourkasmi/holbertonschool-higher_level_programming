# Python Input/Output

## Description
Basic file and user input/output handling in Python.

## Usage
Run scripts with:
```bash
python3 script.py
```

## Example
```python
with open("file.txt", "w") as f:
    f.write("Hello, World!\n")
```

## Testing
Run tests:
```bash
python3 -m doctest ./tests/*
```
